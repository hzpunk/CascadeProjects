# Hzlab Telegram Bot

Telegram бот для уведомления о запуске сайта Hzlab с возможностью хостинга на Vercel.

## Установка

1. Клонируйте репозиторий
2. Установите зависимости: `npm install`
3. Создайте Telegram бота через @BotFather и получите токен
4. Разверните на Vercel

## Переменные окружения

- `BOT_TOKEN` - токен вашего Telegram бота
- `WEBHOOK_URL` - URL для вебхуков (автоматически устанавливается Vercel)

## Команды бота

- `/start` - начать работу с ботом
- `/notify` - получить уведомление о запуске сайта

## Развертывание на Vercel

1. Подключите репозиторий к Vercel
2. Установите переменные окружения в настройках проекта
3. Разверните проект

После развертывания установите вебхук:
```
https://api.telegram.org/bot<YOUR_BOT_TOKEN>/setWebhook?url=https://your-vercel-app.vercel.app/api
```
